Predictor de Candidatos

Información sobre la Plataforma de Evaluación de Candidatos

Esta plataforma permite analizar y predecir la aptitud de candidatos para un puesto de trabajo mediante un modelo de clasificación basado en regresión logística.

¿Cómo funciona?

El sistema trabaja con datos estructurados en formato CSV, los cuales pueden ser subidos por el usuario o generados automáticamente por la plataforma. Cada archivo contiene información clave sobre los candidatos, incluyendo:

  • ID: Identificador único del candidato.
  • Género: Masculino o Femenino.
  • Años de Experiencia: Tiempo trabajado en el área.
  • Nivel Educativo: Primario, secundario o Universitario.
  • Habilidades Técnicas: Competencia en Python, JavaScript y SQL.
  • Apto: Indica si el candidato cumple con los criterios para ser considerado para el puesto.

Proceso de Evaluación

  1. Carga o Generación de Datos: Los usuarios pueden subir su propio archivo CSV o solicitar que el sistema genere uno.
  2. Entrenamiento del Modelo: Primero se visualizan los datos cargados o generados. Luego se aplica un modelo de regresión logística entrenado con los datos ingresados. Se identifican patrones en la experiencia, nivel educativo y habilidades técnicas.
  3. Predicción y Resultados: Una vez entrenado el modelo, el sistema clasifica a los candidatos en Aptos o No Aptos. Los resultados se muestran en la sección de predicciones y resultados.

Beneficios del Sistema

  • Automatización del proceso de selección: Reduce el tiempo de análisis de perfiles.
  • Decisiones basadas en datos: Utiliza criterios objetivos para evaluar candidatos.
  • Flexibilidad en la entrada de datos: Soporta carga de archivos personalizados y generación automática.
  • Interfaz intuitiva: Permite una interacción sencilla con los datos y resultados.

Esta herramienta está diseñada para facilitar la gestión de talento, optimizando la selección de candidatos mediante el uso de machine learning.
